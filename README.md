## üèÅ Starting

```bash
# Clone this project
$ git clone https://github.com/murilorvargas/payment-slip-validator
# Access
$ cd payment-slip-validator
```

### Node

```bash
# Install dependencies
$ yarn
# Run the project
$ yarn dev
# The server will initialize in the <http://localhost:8080>
```

### Docker

```bash
# Run the project
$ docker-compose up -d
# The server will initialize in the <http://localhost:8080>
```
